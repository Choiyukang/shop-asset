# MallBook (쇼핑몰 자산관리) -- 디자인 문서

> Show Me The PRD로 생성됨 (2026-04-14)

## 프로젝트 한 줄 요약
1인 쇼핑몰 사장이 매일의 구매·판매·세금·현금흐름을 **구글시트와 동기화되는 데스크톱 앱**으로 관리하는 무료 자산관리 도구.

## 핵심 결정 요약

| 항목 | 선택 |
|------|------|
| 사용자 | 1인 사장 + 가족/지인 2~3명 |
| 플랫폼 | 데스크톱 앱 (Windows/macOS 더블클릭 설치) |
| 스택 | Tauri 2 + React + SQLite + Google Sheets API |
| 비용 | 0원 (완전 무료, 내 PC 저장) |
| 로그인 | 구글 OAuth (시트 연동용) |
| MVP 범위 | 거래 CRUD + 구글시트 쓰기 + 대시보드 |

## 미결 사항 (종합)

### PRD
- [ ] 부가세 환급률 기준 (일반과세자/간이과세자)
- [ ] 구글시트 동기화 방향 (단방향/양방향) 기본값
- [ ] 앱 이름 최종 확정
- [ ] 로고/아이콘 디자인
- [ ] 데이터 백업 방식 (구글드라이브 자동 백업)

### 데이터 모델
- [ ] 간이과세자 환급 규칙 override 방식
- [ ] 한 거래에 여러 분류 혼합 처리
- [ ] 환율/외화 거래 지원 여부
- [ ] 삭제 정책 (hard vs soft delete)

### 프로젝트 스펙
- [ ] Google Cloud Console 프로젝트 주체 (개인/사업자)
- [ ] 코드 사이닝 인증서 구매 여부
- [ ] 앱 번들 ID / 패키지명 최종
- [ ] 다국어(i18n) 초기 도입 여부
- [ ] 개발 저장소 위치 (GitHub private / 로컬)
